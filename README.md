# Spatiotemporal Forecasting of Air Quality Index in Dhaka

### A Comparative Machine Learning Approach Using 25 Years of Longitudinal Data (2000–2025)

---

## Project Overview

This study analyzes and forecasts the **Air Quality Index (AQI)** in Dhaka, Bangladesh, using a comprehensive 25-year dataset. By leveraging longitudinal data, we compare various Machine Learning models to predict atmospheric pollution patterns, providing insights into urban environmental health.

This project is developed as part of the **Introduction to Data Science** course at the **American International University-Bangladesh (AIUB)**, with the long-term goal of publication in a peer-reviewed environmental or data science conference.

## Dataset Information

The primary dataset used in this research is [`dhaka_air_quality_2000_2025.csv`](https://www.kaggle.com/datasets/shakil10945/dhaka-air-quality-2000-2025-synthetic-dataset).

### Key Features:

- **Temporal Coverage:** 2000 – 2025
- **Objective:** Identify seasonal trends, long-term degradation, and predictive forecasting.
